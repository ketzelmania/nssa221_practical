- Create a pfSense router
- Create a Windows Server 2025
- Create a Windows 11 Client
- Create a Rocky Linux Client

CONFIGURE VMS
- Create a new NIC for the router; set one to LAN
- Create & assign LAN segment to ALL VMs

SET UP AD
- Run ./ADSetup.ps1
- Run ./DhcpSetup.ps1
- Run ./CreateADUser.ps1
- On the Windows client, run ./WindowsClientJoinAD.ps1
- On the Linux client, just set the hostname to <hostname>.gcp1180.com

- Recall: admin account SAM username = gpetitt; password = Password1
- Recall: Windows server hostname = goku; Windows client hostname = winclient; Rocky client hostname = linclient

SET UP RAID
- Command: mdadm --create /dev/md0 --level=5 --raid-devices=3 /dev/nvme0n{2,3,4}
- fstab: lowkey just copy a previous line the settings dont matter

SET UP HTTP SERVER
- dnf install -y openssl httpd mod_ssl
- Modify /etc/httpd/conf/httpd.conf to include some virtual hosts (copy the
  directory relaxation)
- Create virtual host .conf files in /etc/httpd/conf.d/
- Format: <VirtualHost *:80></VirtualHost>
- Fields: DocumentRoot, ServerName, ErrorLog
- For www, the conf file is called _default_.conf

- SSL: create a private key, cert signing request, and a signed cert
- pkey: openssl genrsa -des3 -out private.key 2048
- csr : openssl req -key private.key -new -out server.csr
- cert: openssl x509 -signkey private.key -in server.csr -req -days 365 -out server.crt

- In /etc/httpd/conf.d/ssl.conf, set the server name and key/cert paths

SET UP FTP
- dnf install vsftpd # server
- dnf install lftp # client

- vsftpd config: enable anon access and write perms
- dropbox: create dropbox directory in /var/ftp/ and chgrp to ftp
- set uploaded files to chown root
